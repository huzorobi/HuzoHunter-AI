# Security Policy

HuzoHunter AI is a security product, and we take the security of the project itself
seriously. Thank you for helping keep it and its users safe.

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub Issues.**

Instead, report them privately by emailing:

📧 **huzorobi@gmail.com**

Please include, where possible:

- A description of the vulnerability and its potential impact
- Steps to reproduce or a proof of concept
- Any relevant logs, versions, or configuration details
- Your name/handle if you'd like to be credited

## What to Expect

- We aim to acknowledge reports within **5 business days**.
- We'll keep you informed as we investigate and work on a fix.
- We ask that you give us a reasonable opportunity to remediate before any public
  disclosure (coordinated disclosure).

## Scope

As the project is in an early R&D phase, the codebase and supported configurations are
still evolving. We still welcome reports for any security concern you identify.

## Recognition

With your permission, we're happy to credit researchers who responsibly disclose valid
vulnerabilities.
